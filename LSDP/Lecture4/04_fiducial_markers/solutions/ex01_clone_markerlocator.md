Clone the git repository with the marker tracker.
```
git clone git@github.com:henrikmidtiby/MarkerLocator.git
```

Enter the directory containing the `markertrackerillustrations.py` file and
run the python script.
```
cd MarkerLocator/documentation/pythonpic/
python markertrackerillustrations.py
```

The images generated by the script can now be inspected and compared to the code that 
generated the images.
